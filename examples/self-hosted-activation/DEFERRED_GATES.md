# Deferred Gates

The current product loop is complete only as a self-hosted local activation loop. These gates remain outside this packet:

- Hosted control-plane operation.
- Provider or customer custody proof.
- Spend-window reservation ledger enforcement.
- Cross-org certificate trust registry, JWKS rotation, and revocation.
- Host-specific MCP, browser, shell, network, package-manager, repo, and cloud interception.
- WorkOS/auth.md external attestation.
- Clearing-house or marketplace trust.
